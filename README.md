# Meeting Cost Calculator

This is a simple web project made using HTML, CSS, and JavaScript.  
It helps to calculate the total cost of a meeting based on employees' hourly salaries and meeting duration.

## Features

- Add employee name and hourly salary
- Remove participants if needed
- Enter meeting duration in minutes
- Calculate total meeting cost
- Show cost per minute
- Simple recommendation (whether meeting is worth it or not)
- Save past meetings using localStorage
- View meeting history even after refresh

## How It Works

1. Add employees with their salary per hour  
2. Set meeting duration in minutes  
3. Click calculate button  
4. It shows:
   - Total cost of meeting  
   - Cost per minute  
   - Simple recommendation  
5. All meetings are saved in browser storage

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage (browser storage)


## How to Run the Project

1. Download or clone the project
2. Open folder in VS Code
3. Open `index.html` file in browser
4. Done ✔
